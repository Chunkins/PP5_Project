# PP5_Project
Camera Movement:
forward - W
left - A
backward - S
right - D

down - x
up - space

Toggle Wireframe - RightShift
Switch Model Focus - LeftShift
Reset Focus - R

Directional Light - 0 or 1
Point Light - 2
Spot Light and Directional - 3
Directional and Point Light - 4
